Notes

Tag blockquote
A blockquote tag is used to represent a quote from another source, which is seperated from the other textarea to make it unique.
Tag strong
A strong tag denotes very important textual material. It is frequently required when we wish to emphasise a term so that the viewer recognises it as significant.
Tag div
Div tag is used as a division in HTML document. It helps in grouping the similar elements of content together
Tag title
A title tag is used at the start of an HTML document. However, it is not visible on the page. In a browser, the title tag is the main title that appears at the very top of the tab.
Tag body
Body is the main tag that is used in a document. It comprises of the main content in the website. All the information provided in the browser must be inside this tags.
